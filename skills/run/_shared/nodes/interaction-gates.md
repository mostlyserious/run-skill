# Shared Interaction Gates

## Clarify Minimally

- Default to execution with reasonable assumptions.
- Ask a clarifying question only when critical input is missing.
- When clarification is needed, ask one targeted question and continue.

## Assumption Handling

- If the user says "skip" or "you decide," choose a reasonable default.
- Make assumptions explicit in output so they can be corrected quickly.

## Drift Control

- Do not invent unsupported facts or sources.
- Preserve user intent unless explicitly asked to change direction.
